发表于 2019-01-02 | 分类于 指令 ， Django相关指令

Django / Flask/Tornado

安装
pip check - 检查是否冲突
pip freeze - 检查依赖库版本
pip install
pip install django==1.11 - 更换版本
python -m pip install -U pip – 更新pip

1.通过PyCharm创建项目 –> hellodjango
2..在Terminal的虚拟环境中安装依赖项 –> install django pymysql django-redis djangorestframework pillow
pip check - 重建依赖库
pip freeze > requirements.txt - 依赖库导出
3.创建Django项目 –>django-admin startproject hellodjango . – 点表示当前路径
4.运行项目 –> python manage.py runserver - 启动服务器
http://localhost:8000
http://127.0.0.1:8000
python manage.py runserver 0.0.0.0:8000
5.修改配置文件 –> hellodjango/settings.py
LANGUAGE_CODE = ‘zh-hans’ - 语言设置
TIME_ZONE = ‘Asia/Chongqing’ - 时区设置
DATABASES = {
‘default’: {
‘ENGINE’: ‘django.db.backends.mysql’,
‘NAME’: ‘HRS’,
‘HOST’: ‘47.112.10.147’,
‘PORT’: 3306,
‘USER’: ‘root’,
‘PASSWORD’: ‘123456’,
‘TIME_ZONE’: ‘Asia/Chongqing’
}
}
6.修改init.py文件 –>
import pymysql
pymysql.install_as_MySQLdb()
7.生成Django自带的数据表
python manage.py migrate - 在数据库增加10个表格
8.创建后台管理账号
python manage.py createsuperuser
